# إضافة الآيات الكاملة

لإضافة جميع الآيات الـ6236 آية لجميع السور الـ114، يمكنك:

## الطريقة 1: استخدام API موثوق

استخدم API من مصدر موثوق مثل:
- Tanzil.net API
- Quran.com API
- Alquran.cloud API

## الطريقة 2: إضافة يدوية

أضف الآيات في ملف `data/surahs.ts` لكل سورة:

```typescript
ayahs: [
  { number: 1, text: 'نص الآية...' },
  { number: 2, text: 'نص الآية...' },
  // ... باقي الآيات
]
```

## الطريقة 3: استخدام ملف JSON

يمكن إنشاء ملف JSON منفصل يحتوي على جميع الآيات ثم استيراده.

## ملاحظة مهمة

- تأكد من استخدام نص عثماني موثوق
- تحقق من صحة الآيات
- احترم حقوق النشر
