# كيفية إضافة جميع الآيات

تم إضافة الآيات لسور قصيرة (1, 103, 105, 106, 107, 108, 109, 110, 111, 112, 113, 114).

## لإضافة باقي الآيات:

### الطريقة 1: استخدام API (موصى بها)

استخدم API من مصدر موثوق:

```javascript
// مثال: استخدام Alquran.cloud API
async function fetchAyahs(surahNumber) {
  const response = await fetch(`https://api.alquran.cloud/v1/surah/${surahNumber}/ar.asad`);
  const data = await response.json();
  return data.data.ayahs.map(ayah => ({
    number: ayah.numberInSurah,
    text: ayah.text
  }));
}
```

### الطريقة 2: إضافة يدوية

أضف الآيات في ملف `data/surahs.ts` لكل سورة:

```typescript
{
  number: X,
  // ... باقي البيانات
  ayahs: [
    { number: 1, text: 'نص الآية الأولى...' },
    { number: 2, text: 'نص الآية الثانية...' },
    // ... باقي الآيات
  ],
}
```

### مصادر موثوقة للآيات:

1. **Tanzil.net** - نص عثماني
2. **Alquran.cloud** - API مجاني
3. **Quran.com** - API موثوق
4. **King Fahd Complex** - نص معتمد

### ملاحظات مهمة:

- تأكد من استخدام نص عثماني موثوق
- تحقق من صحة الآيات
- احترم حقوق النشر
- استخدم نفس التنسيق الموجود في السور المضافة
